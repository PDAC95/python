## Requisitos

- Python 3.8+

## Instalacion

1. Clonar el repositorio:
   ```bash
   git clone <url>
   cd nombre-proyecto
   ```

2. Crear entorno virtual:
   ```bash
   python -m venv venv
   ```

3. Activar entorno virtual:
   - Windows: `venv\Scripts\activate`
   - Mac/Linux: `source venv/bin/activate`

4. Instalar dependencias:
   ```bash
   pip install -r requirements.txt
   ```

## Uso

```bash
python src/main.py
```
